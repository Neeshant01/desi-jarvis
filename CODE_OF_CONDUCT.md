# Code of Conduct

This project should stay respectful, practical, and beginner-friendly.

## Expected behavior

- be kind and specific
- explain technical feedback clearly
- help keep the project learnable
- prioritize safe automation behavior

## Unacceptable behavior

- harassment or personal attacks
- harmful automation advice
- spam, trolling, or bad-faith contributions

## Enforcement

The repository owner may remove or moderate contributions that do not support a healthy project environment.
